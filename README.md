## Overview

This project is an **interactive Excel sales analysis dashboard** created to analyze business performance using structured reporting and visual storytelling.  
It is designed to help stakeholders quickly understand key sales metrics such as:

- total revenue
- total profit
- products sold
- products never sold
- profit margin
- monthly and quarterly performance trends
- weekday vs weekend patterns
- country-level filtering and analysis

The workbook combines **analysis** and **dashboard presentation** into a single `.xlsm` file, making it suitable for reporting, portfolio presentation, and business decision support.

## Project Objective

The goal of this project is to demonstrate how Excel can be used as a practical business intelligence tool to:

- cleanly summarize large sales datasets
- build interactive reports for decision-makers
- identify performance trends over time
- compare above-average and below-average performance
- highlight product and revenue opportunities

## Business Value

This dashboard helps answer practical business questions such as:

- Which periods generate the highest revenue and profit?
- Are there products that are not contributing to sales?
